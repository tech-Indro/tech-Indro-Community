## рҹҡҖ Getting Started

### Step 1: Fork the Repository

Click the **Fork** button at the top-right of this repository.

### Step 2: Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/tech-indro.git
cd tech-indro
```

### Step 3: Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### Step 4: Make Your Changes

- Edit files, add content, build projects, or create new resources
- Follow the style and formatting guidelines below

### Step 5: Commit Your Changes

```bash
git add .
git commit -m "Add: clear and descriptive commit message"
```

### Step 6: Push to Your Fork

```bash
git push origin feature/your-feature-name
```

### Step 7: Create a Pull Request

- Go to the original repository
- Click **Compare & pull request**
- Write a clear description of what you changed and why
- Submit your PR
